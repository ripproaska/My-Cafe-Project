## Brew and Books

This is a simple command-line Python application that simulates a simple ordering system for a bookshop and cafe. It supports customer registration, login, guest checkout, employee access, loyalty discounts and stock management all powered by JSON based data storage and modular python functions.

#Feature
- Customer registration, login, and guest checkout
- Loyalty tracking with automatic discount after every 10 purchases
- Employee login with access to stock check and discount application
- Modular codebase split into main.py and functions.py
- Persistent data storage using JSON files
- Role-based interaction and dynamic menu options

# How to Run 
Clone the repository:

Make sure you have Python 3 installed.

Run the application:
python main.py

#Requirement 
No external libraries required built using pythin standard json modul.
